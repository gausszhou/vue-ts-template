# Vue TS Template

## 项目介绍

这是一个基于 Vue 3 + TypeScript + Vite 搭建的项目模板，用于快速开发 Vue 3 项目。

## 项目结构 

```
├── public
├── src
│   ├── assets
│   ├── components
│   ├── views
│   ├── App.vue
│   ├── main.ts
│   ├── router.ts
│   ├── store.ts
│   ├── styles
│   └── utils
├── .eslintrc.js
├── .gitignore
├── index.html
├── package.json
├── pnpm-lock.yaml
├── README.md
├── tsconfig.json
├── tsconfig.node.json
└── vite.config.ts
```
